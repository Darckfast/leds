# Installation

## Python

* Install miniconda Python 2.7
- https://conda.io/miniconda.html

## Node JS
* Install nodejs
https://nodejs.org/en/download/

* Install pug/pugcli
npm install -g pug
npm install -g pugcli

Go to apps\html and type
pug -P -w templates -o ./

It will automatically render all files
